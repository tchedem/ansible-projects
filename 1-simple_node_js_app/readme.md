# Deploy a Node.js app on rockylinux8

In this project, I'll deploy a simple node app on a Linux server using vagrant and ansible


Pour deployer le projet, j'ai eu besoin de :

- Un serveur Linux (rockylinux8)
- node (Install the version define in the project specs)
- npm (gestionnaire de package pour l'environnement d'execution Javascript)


Foreavers Is a tool (app) used to run node script continously


A local domain name. In my case mynodeapp.test


On web http://mynodeapp.test:8080 to acces to the project
